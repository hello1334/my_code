### 알고리즘 로직

* 인기도 순으로 정렬하는 방식으로 구현했습니다.
  * 인기도 필드 명을 'popularity'라고 하고, 내장 메서드 order_by를 활용하여 오름차순으로 정렬하였습니다.
  * 내림차순으로 하기 위해서 해당 값 앞에 음수(-)를 붙여주었습니다.

* 인기도 Top 3 영화의 경우 강조를 위해 알고리즘으로 해당 값을 따로 추출하고, html 인라인 태그로 하이라이트 표시를 해주었습니다.
